# 青色申告 複式簿記 会計アプリ

PRiMPO連携対応の青色申告準拠 会計Webアプリです。  
GitHub Pagesでそのまま動作します（ビルド不要）。

## 機能

- **複式簿記エンジン** — 借方・貸方の二重記帳、貸借バランスチェック
- **青色申告対応** — 青色申告用勘定科目（地代家賃、給料賃金、外注工賃 等）
- **損益計算書（P/L）** — 年次・月次の収益・費用集計
- **総勘定元帳** — 科目別残高・取引履歴
- **消費税管理** — 10%/8%（軽減）/非課税/免税対応、本則課税・簡易課税・免税事業者切替
- **家事按分** — 仕訳ごとに事業割合（%）を設定、自動で事業費/家事費を分離
- **PRiMPO CSV インポート** — 書類スキャンアプリPRiMPOのCSVを直接取り込み
- **CSV エクスポート** — 仕訳帳・損益計算書・消費税集計をCSV出力
- **iPhone 14 Pro Max 最適化** — Safe Area対応、縦スクロールのみ

## GitHub Pagesへのデプロイ手順

1. このリポジトリをGitHubにプッシュ
2. リポジトリの **Settings → Pages**
3. Source: **Deploy from a branch**
4. Branch: `main` / `/ (root)` を選択して **Save**
5. 数分後に `https://<username>.github.io/<repo>/` でアクセス可能

## PRiMPO CSVフォーマット

| 列 | 内容 |
|---|---|
| A | 日付（YYYY-MM-DD） |
| B | 内容・摘要 |
| C | 金額（正=収入、負=支出） |
| D | カテゴリ（任意） |

取込後、科目・税区分を仕訳帳で確認・修正することを推奨します。

## データ保存

ブラウザの `localStorage` にデータを保存します。  
端末・ブラウザを変える場合はCSVエクスポートでバックアップしてください。

## ファイル構成

```
index.html   — メインHTML（画面レイアウト）
style.css    — スタイルシート（iPhone最適化）
accounts.js  — 勘定科目マスタ（青色申告対応）
app.js       — アプリロジック（複式簿記エンジン）
```

## 注意事項

- 本アプリは参考用です。申告の際は税理士に確認してください。
- データはブラウザのlocalStorageに保存されます（サーバー送信なし）。
